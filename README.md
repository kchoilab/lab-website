# IN/SB Lab Website

Source code for the **Integrative Neuro-Systems Biology (IN/SB) Lab** website.

**Live site:** [kchoilab.com](https://kchoilab.com)

Built with [Jekyll](https://jekyllrb.com/) using the [al-folio](https://github.com/alshedivat/al-folio) theme, hosted on GitHub Pages.

---

## Site Structure

| Directory / File         | Purpose                                      |
| ------------------------ | -------------------------------------------- |
| `_config.yml`            | Main site configuration                      |
| `_pages/`                | Top-level pages (home, people, research, …)  |
| `_posts/`                | News posts                                   |
| `_projects/`             | Research project entries                     |
| `_bibliography/`         | BibTeX references for the Publications page  |
| `_data/`                 | Structured data (e.g., coauthors, venues)    |
| `assets/`                | Images, PDFs, and other static files         |
| `_includes/` / `_layouts/` | Liquid templates                           |

## Pages

- **Home** — Lab overview and latest news
- **People** — Lab members and join-us information
- **Research** — Research directions
- **Publications** — Paper list (auto-generated from BibTeX)
- **Teaching** — Courses taught by lab members
- **Software** — Open-source tools developed by the lab
- **News** — Lab news and updates
- **Contact** — Contact information
